Platform Overview

We Are The Change is a revolutionary civic engagement platform designed to bridge the gap between citizens and municipal services in Bahrain. Our platform transforms how communities interact with local authorities and participate in volunteer activities.

Core Mission:

  • Empower citizens to report local issues efficiently
  • Facilitate volunteer coordination for community projects
  • Provide transparent tracking of issue resolution
  • Enhance community participation through technology

Technical Architecture:

  • Frontend: Built with FlutterFlow for cross-platform compatibility
  • Backend: Powered by Firebase for real-time data synchronization
  • Database: Cloud Firestore for scalable data storage
  • Storage: Firebase Storage for secure media handling
  • Authentication: Firebase Auth with OTP verification

Download & Installation

Getting started with We Are The Change is simple. Choose from these options:

Option 1: Android APK (Recommended)

  • Direct download from our website
  • File size: Approximately 25 MB
  • Requirements: Android 8.0 (Oreo) or higher
  • Installation: Allow installation from unknown sources
  • Updates: Manual updates through our website

Option 2: Live Demo (Instant Access)

  • Accessible via web browser at: https://change.flutterflow.app
  • No installation required
  • Works on all modern browsers
  • Mobile-responsive design
  • Limited functionality compared to full app

Installation Steps for Android:

  1. Download the APK file from our website
  2. Enable "Install from unknown sources" in settings
  3. Locate the downloaded file in your file manager
  4. Tap the file to begin installation
  5. Follow on-screen instructions
  6. Launch the app and create your account

Security Implementation

Security is our top priority. We've implemented multiple layers of protection:

1. Authentication & Authorization:

  • Firebase Authentication with email/password
  • One-Time Password (OTP) verification
  • Role-based access control (RBAC)
  • Session management with automatic logout
  • Password complexity requirements

2. Data Protection:

  • End-to-end encryption for sensitive data
  • SSL/TLS for all communications
  • Secure Firebase database rules
  • Regular data backup and recovery procedures
  • GDPR-compliant data handling

3. Methodology Implementation:

  • SQUARE Methodology for security requirements
  • STRIDE threat modeling for risk assessment
  • Regular penetration testing
  • Vulnerability assessment and management
  • Security incident response plan

4. Media & File Security:

  • Secure file upload with virus scanning
  • Image compression and validation
  • Access control for uploaded files
  • Automatic cleanup of temporary files
  • Secure storage with Firebase rules
